function drawSkyline(x, y, s) {
    push();

    translate(x, y);

    scale(s);

    fill('rgb(150,0,0)');

    stroke('rgb(255, 255, 255)');

    strokeWeight(3);

    beginShape();
    vertex(0, -40);
    vertex(3, -40);
    vertex(3, -35);
    vertex(10, -35);
    vertex(10, -39);
    vertex(13, -39);
    vertex(13, -49);
    vertex(11, -49);
    endShape();
    pop();
}
An example of an abstraction in my program is the drawSkyline() function. This simplifies my code because it is one specific function does only one thing and not everything piece of the program. I created this drawSkyline() function to use parameter. The given parameter are (x,y,s) which is the x-position, y-position, and a specific scaled ratio. This is abstraction is useful because incase i need to change the position of the city skyline or I need to make the city smaller or bigger I do not need to look through a bunch of messy code.
